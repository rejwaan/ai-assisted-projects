# AI-Assisted Projects

The main purpose of this repository is to explore how AI tools can be used effectively in real-world software development, while still focusing on learning core programming and frontend/backend fundamentals.

## 🚀 What This Repo Contains

- AI-assisted frontend projects (HTML, CSS, JavaScript)
- Small utility apps and experiments
- Portfolio-related projects
- Learning-based implementations with AI guidance
- Modified and improved versions of AI-generated ideas

## 🧠 Learning Approach

- Focus on learning, not just copying
- Improve problem-solving with AI support

## 🛠 Example Projects
